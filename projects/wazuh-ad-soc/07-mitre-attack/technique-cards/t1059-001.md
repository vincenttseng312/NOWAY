# T1059.001 PowerShell

## 技術說明
以 PowerShell 執行攻擊命令/腳本；為 T1059 的子技術。以官方為準。

## 在本專題中的表現
靶機上可疑 PowerShell 執行，尤其父程序為 Office、含編碼命令或下載行為。

## 可觀測資料
4688（命令列）、4104 Script Block Logging（需啟用）；見 [[evt-powershell-suspicious]]。可疑旗標清單見父層 [[lolbin-and-powershell-abuse]]。

## 相關情境與事件
[[scn-suspicious-powershell]]。

## 偵測與處置要點
命令列旗標 > 程序名、父程序鏈、是否下載/記憶體執行；建議啟用 Script Block Logging。
